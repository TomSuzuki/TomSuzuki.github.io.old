Gyro STG

説明

大学の講義で制作した課題のゲームです。
コントローラとゲームの両方をグループで制作しました。

詳細

  • LINK: https://github.com/TomSuzuki/Arduino_Game
  • 制作時期：2020/01

　制御と状態の取得はArduino、取得した情報をprocessingに送りゲームをします。コントローラには加速度センサ、スイッチ、LEDライト、振動モータ、LCDディスプレイを搭載しています。学期末で他の課題も多い時期に短い期間で制作したため作りたかった目標を完全には達成することができませんでした。

使用した技術

  • Arduino
  • C
  • Processing3
  • Java
